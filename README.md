# Portfolyo

Kişisel portfolyo sitem. Next.js, Tailwind, Three.js ve GSAP ile yazıldı.

## Çalıştırma

```bash
npm install
npm run dev
```

http://localhost:3000

## Deploy

Vercel'e bağlı, `main`'e her push otomatik yayınlanıyor. Canlı adres `NEXT_PUBLIC_SITE_URL` ortam değişkeninden okunur (sitemap, robots ve OG etiketleri için).
